JavaScript - Brendan Eich

JavaScript - bu dasturlash tili bo'lib, asason web-sahifalarda interaktivlik yaratish uchun ishlatiladi. U 1995 yilda Brendan Eich tomonidan Netscape kompaniyasida ishlab chiqilgan Dastlab "Mocha" deb nomlangan,keyin "LiveScript"va nihoyat "JavaScript" deb o'zgartirildi

Tarixi va rivojlanishi

1995: Brendan Eich JavaScriptni ishlab chiqadi

1996: JavaScript standartlashtiriladi va ECMAScript nomi bilan tanilgan,

2005-yillar: AJAX texnologiyalari bilan JavaScripning ommolashuvi boshlanadi.

2015: ECMAScript 6 (ES6) chiqariladi, bu JavaScripning katta yangilanishidir va yangi sintaksis va funksuyalarini kiritadi.

Foydalanish joylari

Veb-sahifalar: Interaktiv elementlar yaratish uchun (tugmalar, formalar, animatsiyalar).

Server: Node.js orqali server kodlari yozish.

Mobil Ilovalar: React Native yordamida mobil ilovalar yaratish.

Analitika: Foydalanuvchi harakatlarini kuzatish va tahlil qilish.

Math Object

JavaScript'da Math object matematik funksiyalar va konstantalar bilan ishlash uchun mo'ljallangan. U veb ilovalarida ko'plab matematik amallarni bajarishga yordam beradi. Bu ob'ekt statik bo'lib, Math obyekti bilan to'g'ridan-to'g'ri ishlanadi.

Math.PI

Pi qiymatini ifodalaydi, taxminan 3.14159 ga teng.

console.log(Math.PI); // 3.141592653589793

Math.round()

PSonni eng yaqin butunga aylantiradi.

console.log(Math.round(4.7)); // 5

console.log(Math.round(4.4)); // 4

Math.ceil()

Sonni yuqoriga qarab butunga aylantiradi.

console.log(Math.ceil(4.1)); // 5

Math.floor()

Sonni pastga qarab butunga aylantiradi.

console.log(Math.floor(4.9)); // 4

Math.random()

0 va 1 oralig'ida tasodifiy son yaratadi.

console.log(Math.random()); // 4

Math.max()

Bir nechta argumentlar orasidan eng katta qiymatni qaytaradi.

console.log(Math.max(1, 3, 2)); // 3

Math.min()

Bir nechta argumentlar orasidan eng kichik qiymatni qaytaradi.

console.log(Math.min(1, 3, 2)); // 1

Math.pow()

Bir sonni ikkinchisiga daraja qilish uchun ishlatiladi.

console.log(Math.pow(2, 3)); // 8 (2 darajasi 3)

Math.sqrt()

Sonning kvadrat ildizini qaytaradi.

console.log(Math.sqrt(16)); // 4

Math.abs()

Sonning modulini (musbat qiymat) qaytaradi.

console.log(Math.abs(-5)); // 5

Math.log()

Natural logarifmni qaytaradi (sonning tabiiy logarifmi).

console.log(Math.log(Math.E)); // 1

console.log(Math.log(1)); // 0

Math.exp()

e ni ko'rsatilgan darajaga ko'taradi.

console.log(Math.exp(1)); // 2.718281828459045 (e)

Data Types

Primitive

String

String: Matnli ma'lumotlarni ifodalaydi. Ikkita qo'shtirnoq yoki bitta qo'shtirnoq ichida yoziladi.

let greeting = "Hello, World!";

Number

Number: Sonli ma'lumotlarni ifodalaydi. Butun sonlar va o'nlik sonlarni o'z ichiga oladi.

let age = 25;

let price = 19.99;

Boolean

Boolean: True yoki False qiymatlarini ifodalaydi. Mantiqiy operatsiyalarda qo'llaniladi.

let isValid = true;

let hasPermission = false;

Undefined

Undefined: Qiymat aniqlanmagan yoki o'zgaruvchi e'lon qilinganda lekin qiymat belgilamaganda bo'ladi.

let x;

console.log(x); // undefined

Symbol

Symbol: Unikal va o'zgarmas qiymatni ifodalaydi. Ko'pincha identifier sifatida ishlatiladi.

let sym = Symbol('description');

BigInt

BigInt: Juda katta butun sonlarni ifodalash uchun ishlatiladi.

let bigNumber = BigInt (9007199254740991);

Non-Primitive

Object

Murakkab ma'lumotlarni ifodalaydi va o'z ichiga ma'lumotlar to'plamini oladi.

let person = {
name: "John",
age: 30,
isStudent: True};

Array

Array: Qiymatlar ro'yxatini ifodalaydi. JavaScript'da bu shuningdek ob'ektning bir turi.

let colors = ["red","green","blue";

Function

Function: Funksiyalar kodlarni takrorlash yoki qayta ishlash uchun ishlatiladi.

function greet() {
console.log("Hello world");
}
greet();

Date

Date: Sana va vaqt bilan ishlash uchun ishlatiladi.

let today = new Date();

IF ELSE

JavaScript'da if...else shartlari mantiqiy operatsiyalarni bajarishda juda muhim rol o'ynaydi. Bu shartlar kodni muayyan sharoitlar asosida ishlatish imkonini beradi.

IF

IF sharti faqatgina shart to'g'ri bo'lganda bajariladi.

let age = 18;
if (age >= 18) {
console.log("Siz yetuk yoshdasiz.");
}

Info:Agar age qiymati 18 yoki undan katta bo'lsa, "Siz yetuk yoshdasiz" xabari chiqariladi.

ELSE

IF sharti faqatgina shart to'g'ri bo'lganda bajariladi.

let age = 16;
if (age >= 18) {
console.log("Siz yetuk yoshdasiz.");
} else {
console.log("Siz yetuk yoshda emassiz.");
}

Info:Agar age qiymati 18 dan kichik bo'lsa, "Siz yetuk yoshda emassiz" xabari chiqariladi.

ELSE IF

ELSE IF sharti qo'shimcha shartlarni tekshirish uchun ishlatiladi.

let age = 20;
if (age < 13) {
console.log("Siz bola yoshdasiz."
else if (age >= 13 && age < 18) {
console.log("Siz o'smir yoshdasiz.");
} else {
console.log("Siz yetuk yoshdasiz.");

Info:Bu yerda turli yosh guruhlari bo'yicha xabarlar chiqariladi: bola, o'smir yoki yetuk yosh.

Nested IF Shartlari

IF shartlari ichida boshqa IF shartlarini ishlatish mumkin.

let age = 25;
if (age >= 18) {
console.log("Siz yetuk yoshdasiz.");
if (age >= 21) {
console.log("Siz ovoz berish huquqiga egasiz.");
}
} else {
console.log("Siz yetuk yoshda emassiz.");
}

Info:Bu kodda birinchi if sharti ichida boshqa if sharti ishlatiladi.

Ternary Operator

Yana bir qulay usul â€“ ternary operatori. Bu oddiy shartlarni yozishda ishlatiladi.

let age = 18;
let message = age >= 18 ? "Siz yetuk yoshdasiz." : "Siz yetuk yoshda emassiz.";
console.log(message);

Info:Shart to'g'ri bo'lsa, birinchi xabar, noto'g'ri bo'lsa, ikkinchi xabar chiqariladi.

LOOP

Loop - bu dasturlashda biror bir kod qatorini takroran bajarish uchun ishlatiladigan konstruktsiya. Loop yordamida biz bir xil kodni bir necha marta takrorlashimiz mumkin, bu esa kodni soddalashtiradi va ortiqcha kod yozishni kamaytiradi.Looplarning asosiy turlari - for, while, va do-while looplaridir.

For Loop

For loop - bu belgilangan sonli iteratsiyalar uchun ishlatiladi, ya'ni oldindan aniq qancha marta bajarilishi kerakligini bilgan holatlarda foydalanamiz.

for (let i = 0; i < 5; i++) {
console.log(i);
}

Tushuntirish

let i = 0 - boshlang'ich qiymat (i 0 dan boshlanadi).

i < 5 - shart (i 5 dan kichik bo'lganida loop ishlaydi).

i++ - increment (i ning qiymatini har safar 1 ga oshiradi).

console.log(i) - har safar loop ishlaganda i ning qiymatini chiqaradi.

Natija: 0, 1, 2, 3, 4.

While Loop

While loop - bu shart to'g'ri ekan davom etadigan loop. Shart noto'g'ri bo'lganda loop tugaydi.

let i = 0;
while (i < 5) {
console.log(i);
i++;
}

Tushuntirish

let i = 0 - boshlang'ich qiymat (i 0 dan boshlanadi).

i < 5 - shart (i 5 dan kichik bo'lganida loop ishlaydi).

console.log(i) - har safar loop ishlaganda i ning qiymatini chiqaradi.

i++ - har safar i ning qiymatini 1 ga oshiradi.

Natija: 0, 1, 2, 3, 4.

Do-While Loop

Do-while loop - bu kamida bir marta bajariladi, keyin shartni tekshiradi. Agar shart to'g'ri bo'lsa, takrorlanadi.

let i = 0;
do {
console.log(i);
i++;
} while (i < 5);

Tushuntirish

let i = 0 - boshlang'ich qiymat (i 0 dan boshlanadi).

console.log(i) - har safar loop ishlaganda i ning qiymatini chiqaradi.

i++ - har safar i ning qiymatini 1 ga oshiradi.

while (i < 5) - shart (i 5 dan kichik bo'lganida loop davom etadi).

Natija: 0, 1, 2, 3, 4.

Function

Function - bu dasturlashning asosiy tushunchalaridan biri bo'lib, takrorlanadigan kod qatorlarini bir joyda to'plab, ularni nom bilan chaqirish imkonini beradi. Bu kodni soddalashtiradi va qayta ishlatish imkoniyatini beradi.JavaScriptda funksiyalarni turli xil shakllarda yaratish va ishlatish mumkin. Funksiyalarning asosiy turlari quyidagilardir:

Standart Funksiya Deklaratsiyasi (Function Declaration)

Bu usul yordamida funksiya yaratiladi va uni funksiya nomi orqali chaqirish mumkin.

function yigindi(a, b) {
return a + b;
}
console.log(yigindi(3, 5)); // 8

Tushuntirish

function yigindi(a, b): Funksiya yigindi deb nomlanadi va ikkita parametr qabul qiladi: a va b.

return a + b: Funksiya a va b ning yig'indisini qaytaradi.

console.log(yigindi(3, 5)): Funksiyani chaqirib, natijani konsolga chiqaramiz. Natija: 8.

Funksiya Ifodasi (Function Expression)

Bu usul yordamida funksiya yaratiladi va uni o'zgaruvchi orqali chaqirish mumkin.

let kvadrat = function(x) {
return x * x;
};
console.log(kvadrat(4)); // 16

Tushuntirish

let kvadrat = function(x): kvadrat nomli o'zgaruvchiga anonim (nomlanmagan) funksiya tayinlanadi.

return x * x: Funksiya x ning kvadratini qaytaradi.

console.log(kvadrat(4)): Funksiyani chaqirib, natijani konsolga chiqaramiz. Natija: 16.

Anonim Funksiya (Anonymous Function)

Anonim funksiya - bu nomga ega bo'lmagan funksiya. Ular ko'pincha funksiya ifodalarida ishlatiladi yoki boshqa funksiyalarga uzatiladi.

let sum = function(a, b) {
return a + b;
};
console.log(sum(3, 5)); // 8

Tushuntirish

let yigindi = function(a, b): yigindi nomli o'zgaruvchiga anonim (nomlanmagan) funksiya tayinlanadi.

return a + b: Funksiya a va b ning yig'indisini qaytaradi.

console.log(yigindi(3, 5)): Funksiyani chaqirib, natijani konsolga chiqaramiz. Natija: 8.

Arrow Funksiya (Arrow Function)

ES6 da kiritilgan va qisqaroq sintaksisga ega bo'lgan funksiya. Arrow funksiyalar bu funksiyalarning this kalit so'zini qanday ishlatishini o'zgartiradi.

let kub = (x) => {
return x * x * x;
};
console.log(kub(3)); // 27

Tushuntirish

let kub = (x) => { return x * x * x; }: kub nomli o'zgaruvchiga x parametrini qabul qiluvchi arrow funksiya tayinlanadi.

return x * x * x: Funksiya x ning kubini qaytaradi.

console.log(kub(3)): Funksiyani chaqirib, natijani konsolga chiqaramiz. Natija: 27.

Ichki Funksiyalar (Inner Functions)

Funksiyalar boshqa funksiyalar ichida yaratilishi mumkin. Bu funksiyalar faqat tashqi funksiyalar ichida mavjud bo'ladi.

function tashqi() {
function ichki() {
console.log("Ichki funksiya");
}
ichki();
}
tashqi(); // "Ichki funksiya"

Tushuntirish

function tashqi(): tashqi nomli funksiya yaratiladi.

function ichki(): ichki nomli ichki funksiya tashqi funksiyasi ichida yaratiladi.

ichki(): Ichki funksiya chaqiriladi va "Ichki funksiya" matni konsolga chiqariladi.

O'z-o'zini chaqiruvchi Funksiyalar (Immediately Invoked Function Expressions - IIFE)

Bu funksiyalar darhol yaratilgandan so'ng chaqiriladi.

(function() {
console.log("Darhol chaqirilgan funksiya");
})();

Tushuntirish

(function() { ... }): Anonim funksiya yaratiladi.

(...)(): Funksiya darhol chaqiriladi va "Darhol chaqirilgan funksiya" matni konsolga chiqariladi.

Generator Funksiyalar (Generator Functions)

Generator funksiyalar * belgisidan foydalanib yaratiladi va yield kalit so'zini ishlatadi. Bu funksiyalarni next() metodi yordamida to'xtatish va qayta boshlash mumkin.

function* generator() {
yield 1;
yield 2;
yield 3;
}
let gen = generator();
console.log(gen.next().value); // 1
console.log(gen.next().value); // 2
console.log(gen.next().value); // 3

Tushuntirish

function* generator(): generator nomli generator funksiya yaratiladi.

yield: Funksiya qiymatni qaytaradi va keyingi chaqiruvni kutadi.

let gen = generator(): Generator funksiyasini chaqiramiz.

gen.next().value: Generator funksiyasining keyingi qiymatini qaytaradi.

Date

Date obyekti JavaScriptda vaqt va sana bilan ishlash uchun ishlatiladi. Bu obyekt yordamida hozirgi vaqtni olish, sanalar orasidagi farqni hisoblash va sanalarni manipulyatsiya qilish mumkin.

DATE obyektini yaratish

JavaScriptda Date obyektini yaratish uchun new Date() konstruktoridan foydalanamiz:

let hozirgiSana = new Date();
console.log(hozirgiSana);

Bu kod hozirgi sana va vaqtni qaytaradi.

Parametrli DATE obyektini yaratish

Yil, oy, kun va boshqalar bilan yaratish:

let tugilganKun = new Date(1990, 5, 15);
// 1990 yil 15 iyun (oylar 0 dan boshlab sanaladi: 0 - yanvar, 5 - iyun)
console.log(tugilganKun);

Mavjud sanadan milisekundlar:

let milisekund = new Date(86400000);
// 1970 yil 2 yanvar (1 kun = 86400000 milisekund) console.log(milisekund);

Sana va vaqt qatoridan yaratish:

let sanaVaqt = new Date("2024-11-07T13:51:00"); console.log(sanaVaqt);

DATE obyektlaridan ma'lumot olish

Date obyekti yordamida sananing turli qismlarini olish mumkin:

let hozirgiSana = new Date();
console.log(hozirgiSana.getFullYear()); // Yilni qaytaradi
console.log(hozirgiSana.getMonth()); // Oy (0 - yanvar, 11 - dekabr)
console.log(hozirgiSana.getDate()); // Kun (1 - 31)
console.log(hozirgiSana.getDay()); // Haftaning kuni (0 - yakshanba, 6 - shanba)
console.log(hozirgiSana.getHours()); // Soat (0 - 23)
console.log(hozirgiSana.getMinutes()); // Daqiqa (0 - 59)
console.log(hozirgiSana.getSeconds()); // Sekund (0 - 59)

DATE obyektlarini manipulyatsiya qilish

Date obyektidagi sanani o'zgartirish uchun quyidagi metodlardan foydalanamiz:

let sana = new Date();
sana.setFullYear(2025);
sana.setMonth(11); // Dekabr
sana.setDate(25);
sana.setHours(10);
sana.setMinutes(30);
sana.setSeconds(45);
console.log(sana); // Natijada: 2025-12-25T10:30:45

Sanalar orasidagi farqni hisoblash

Sanalar orasidagi farqni hisoblash uchun milisekundlardan foydalanamiz:

let sana1 = new Date('2024-01-01');
let sana2 = new Date('2024-12-31');
let farq = sana2 - sana1;
let kunFarqi = farq / (1000 * 60 * 60 * 24); // Milisekundlardan kunga o'girish
console.log(`Sanalar orasidagi farq: ${kunFarqi} kun`);
